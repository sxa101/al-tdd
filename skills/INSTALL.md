# Installing the AL-TDD Skill

## 1. Gemini CLI Installation

To install this skill for the Gemini CLI, copy the `skills/al_tdd` folder to your local skills directory.

```bash
# Example: Copying to the default skills location on Windows
xcopy /E /I "skills\al_tdd" "%USERPROFILE%\.gemini\skills\al_tdd"
```

## 2. Using the Skill

Once installed, the skill instructions will be available to the agent. You can explicitly invoke the methodology by saying:

> "Initialize this project using AL-TDD"

Or by simple referencing "AL-TDD" in your request.
